# 스파르타 던전 (Text 게임) 만들기
#### 참여인원 밑 역할
* 박건오 - 전투, 몬스터, 던전
* 유호진 - 미니게임(카지노 > 경마, 블랙잭)
* 윤재현(팀장) - 상점, 도전과제, 인사평가 
* 이주환 - 전투, 스킬
* 최보훈 - 플레이어, 상태관리
#### 컨밴션 규칙
type
- 코드수정 [Modify]
- 추가 [Add]
- 오류해결 [Fix]
  
body - 내용적기
예시) [Add] 휴식 기능 추가
